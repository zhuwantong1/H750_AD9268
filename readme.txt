/****************************************/
这是AD9268的单片机驱动程序，这个程序可以学习spi的发送和读取，AD9268的寄存器配置，并转串操作，定时器产生pwm，ns级延时，以及中断捕获操作
/****************************************/
有缺陷的地方是：ad不能发挥最大速率，因为单片机的定时器产生pwm波的时候，tim1和tim2的频率会有相互影响。
当tim1为2M时，tim2可以为40M，但是tim1为40M时，tim2不能为2M。同时捕获时，会有pwm丢失的问题（此时在捕获的中断里没有关闭pwm）。而且没办法同时将两个定时器的速率都配置的很大，会有一个PWM不输出。所以选择只捕获一个ADC_CLK，然后进行并转串处理，等74_CLK16个周期处理完，再捕获ADC_CLK，依次循环，这样能采数据，但是速率很慢。
/****************************************/
当进行GPIO的高低电平的读取时，第一位有概率会出错。给再PL上升沿之后加了一个延时，解决了这个问题。但是当74_CLK频率变化时，延时有可能会出现新错误。

添加readme文件，修改ans的变量类型，得到正确结果，0V时不会再有变到65535的情况。
VREF是寄存器配置2.0V的电压，0X18寄存器默认值为0XC0。
